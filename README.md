# Sentiment and Sarcasm Classification with BESSTIE

This project compares classical machine learning models and BERT-base for sentiment classification and sarcasm detection using the BESSTIE dataset.

## Project Tasks

1. Sentiment classification
2. Sarcasm classification

## Models

- Majority Class Baseline
- TF-IDF + Logistic Regression
- TF-IDF + Linear SVM
- BERT-base-uncased

## Repository Structure

- notebooks/: Jupyter notebooks for experiments
- src/: reusable helper code
- outputs/: generated tables, figures, predictions, and models
- app/: Gradio demo
- report/: final report materials
- slides/: final presentation
